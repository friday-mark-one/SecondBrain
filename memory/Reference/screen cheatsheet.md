- screen - Start screen
- Ctrl + a is mandatory for all commands
- v - version
- ? - list of commands
- c - new screen
- n - next screen
- p - previous screen
- d - detach the screen
- screen -ls  - list all detached screen sessions
- screen -r - reattach to a screen session
- screen -r <id> - to reattach to a particlular screen session
- x - lock screen with ubuntu login password
- k - kill screen
- ctrl-a + ctrl a - flipping between two same window
- <num> - to switch to a particluar screen
- w - show active screens (current screen is marked with an astrerisk *)
- " - list all screen sessions (j and k to move up and down)
- A - name screen sessions
- \ - quit all screen sessions
- [ - copy mode on (enter to start and stop)
- ] - paste copied text
- _ - monitor for silence
- M - monitor for activity
- S - split screen horizontally
- [tab] - to switch to split tab
- X - kill current split region (only the view)
- | - vertical split
- Q - destroy all split view except current
